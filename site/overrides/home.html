Get Up and Running in Minutes

1

Install the Client

go install github.com/nxrvl/nxpose/cmd/client@latest
2

Register with a Server

nxpose register --server your-server.com:8443
3

Expose Your Service

nxpose expose http 8080
# Your service is now available at https://abc123.your-server.com
